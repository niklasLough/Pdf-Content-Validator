File Validator Help Page

Details on how to use the app

Welcome to the help page for the PDF Validator App.

This app validates the presence of specific elements within a PDF file. The application checks if a given string (e.g., "Gesamtpreis") and a corresponding value (e.g., 4.1300,00â‚¬) appear on the same line within the PDF.

There are two ways to use the app:

  1. Upload a PDF file and enter the keyword and value you want to validate.
  2. Upload a CSV file containing multiple keywords and values to validate at once.

Here is a step-by-step guide on how to use the home app:

  1. Upload a PDF file using the form on the home page.
  2. Enter the keyword and value you want to validate.
  3. Click the submit button.
  4. See the results as to if the inputs appear on the same line in the PDF. The location of the values is highlighted in the PDF shown.
  5. If the document is a travel confirmation a result of the total price validation will also be shown.

Here step-by-step guide on how to use the CSV app:

  1. Upload a PDF file using the form on the home page.
  2. Upload a CSV file of keyword and value pairs.
  3. Click the submit button.
  4. See the results in the table as to if the inputs appear on the same line in the PDF. The location of the values are highlighted in the PDF shown.
