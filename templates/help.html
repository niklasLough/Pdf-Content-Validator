Details on how to use the app

Welcome to the help page for the TUI PDF Validator App.

This app validates the presence of specific elements within a PDF file. The application checks if a given string (e.g., "Gesamtpreis") and a corresponding value (e.g., 4.1300,00â‚¬) appear on the same line within the PDF.

Here are the steps to use the app:

  1. Upload a PDF file using the form on the home page.
  2. Enter the keyword and value you want to validate.
  3. Click the submit button.
  4. See the results as to if the inputs appear on the same line.
